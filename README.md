### DEV

1. Clonar proyecto
2. Copiar el

```
env.template
```

y renombrar a

```
.env
```

3. ejecutar

```
npm i
```

4. levantar imagen docker (docker desktop)

```
   docker-compose up -d
```

5. levantar backend de nest

```
nest start:dev
```

6. ir a la url del server

```
localhost:3000/graphql
```
